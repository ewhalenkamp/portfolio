Project Description

This portfolio features four projects, including a YouTube API-fetching music player, a gradebook, a proprietary API-fetching Pokédex website, and a nonogram puzzle solver. Of these, only the puzzle solver and the gradebook were made before construction started on this project.

While the portfolio itself is a mere compilation, the Pokédex and music player (and most of the technologies required to create them, including React and Springboot) were learned from scratch in the few weeks since I've started crafting this portfolio. I used Webpack to serve a prototype of RecordWall, though this is no longer the case as RecordWall has released on the App Store.
